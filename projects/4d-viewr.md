# 4D VIEWR – AR Interaction Platform

**Context:** HFC Media  
**Stack:** Unity, C#, AR frameworks (Vuforia / ARCore), cloud services  
**Focus:** Augmented Reality, Interactive Systems  

---

## Overview

Developed an AR application designed to create a “holographic internet” experience, where physical objects trigger interactive digital content.

The system used image tracking to instantiate interactive 3D scenes with dynamic content and user interaction.

---

## Problem

- Needed a reliable way to anchor digital content to real-world objects  
- Required scalable content delivery without bundling all assets locally  
- Interaction needed to feel intuitive within AR space  
- Limited tools existed for combining AR with dynamic data sources  

---

## Core System

- Image-based tracking using physical markers (stickers, coasters)  
- Dynamic scene instantiation using cloud-hosted asset bundles  
- Modular scene design with interactive components  

---

## Interaction Design

- World-space UI elements (buttons, menus)  
- Scene transitions triggered by user interaction  
- Ground-plane detection to reposition content into the environment  

---

## Additional Features

- AR + GPS integration for location-based experiences  
- Integration with external APIs (weather, news) to populate scenes  
- Real-time content updates through cloud services  

---

## Technical Details

- Asset bundle loading for scalable content delivery  
- Event-driven interaction system for AR UI  
- Integration of multiple APIs into AR scenes  
- Hybrid tracking (image-based + ground-plane)  

---

## Results / Impact

- Delivered a fully interactive AR content platform  
- Demonstrated scalable approach to AR content distribution  
- Enabled dynamic, data-driven AR experiences
